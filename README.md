# Daily Habit Tracker

A simple web application built with **Flask** and tested with **Selenium** to help track daily habits.

---

## 🚀 Features
- Add and tick off daily habits
- Track streaks and completion percentage
- Automated testing with Selenium
- Clean UI with HTML/CSS

---

## ⚙️ Setup Instructions

1. **Clone the repository**
   ```bash
   git clone https://github.com/Chethan-V-5115/habit-tracker.git
   cd habit-tracker

2. Create a virtual environment
Bash

python -m venv .venv
.venv\Scripts\activate

3. Install dependencies
Bash

pip install flask selenium


4. Run the Flask app
Bash

python app.py

Open browser at: http://127.0.0.1:5000

5.Run Selenium tests
Bash

python test_habits.py







Project Structure



habit-tracker/
│── app.py              # Flask app
│── test_habits.py      # Selenium tests
│── habits.json         # Data storage
│── templates/          # HTML templates
│── static/             # CSS styles
